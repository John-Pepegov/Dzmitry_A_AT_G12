Пара заданий на ввод/вывод, каждое из заданий сделать в отдельном классе с собственным main (в том числе, на каждый объект класса из пакета playground.essence), в котором вызываем нестатический невозвратный метод этого же класса, который:
 + выводит в консоль фразу "Hello, I just got 'X' from you!", где Х - фраза, вводимая в консоль с клавиатуры.
 + записывает в файл фразу "Hello, I just got 'X' from you!", где Х - фраза, вводимая в консоль с клавиатуры.
 + читает текст из файла, убирает из него все гласные и записывает новый текст в тот же файл, затирая оригинал.
 + читает текст из файла, убирает из него все согласные и записывает новый текст в тот же файл в конец, через пустую строку ниже оригинала.
 - читает текст из файла, считает в тексте количество символов и выводит результат в консоль.
 - читает текст из файла, считает в тексте количество символов и записывает результат в новый файл, имя которого строится по маске текущая_дата_время_количество.
 - создает какой-нибудь объект класса (добавить имплементацию интерфейса Serializable) из пакета playground.essence и записывает его в файл
 - читает какой-нибудь объект класса (добавить имплементацию интерфейса Serializable) из пакета playground.essence из файла (например, записанный в предыдущем задании) и записывает в другой текстовый файл результат возвращаемого значения метода toString() этого обьекта
